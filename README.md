# 🍥 Naruto Dating System

Projeto experimental desenvolvido para realizar um pedido de namoro temático inspirado no universo Naruto.

## 📖 Sobre o Projeto

Após anos estudando automação, scripts, infraestrutura, cibersegurança e inteligência artificial, surgiu a seguinte pergunta:

> "E se eu utilizasse minhas habilidades técnicas para resolver um problema sentimental?"

A resposta foi este projeto.

## 🎯 Objetivo

Criar uma experiência personalizada baseada em Naruto para realizar um pedido de namoro de forma criativa e divertida.

## ⚙️ Tecnologias Utilizadas

* Python
* HTML/CSS
* Persistência emocional não documentada
* Quantidades excessivas de esperança
